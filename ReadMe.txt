Структура Cmd - хранит токен(слово) одной команды и указатель next, явл частью списка
Структура TableRedirect - хранит 
Структура ProgForRun - хранит :
                - указатель на указатель на структуру Cmd
                - указатель на указатель на структуру TableRedirect
                - указтель на массив тип команды




Разбор AutoCompletion - получает по одному слову и выполняются функции ниже
IfControlCharacters - функция проверяет наличие управляющих символов и выполняет 
    действия если такие символы есть
CountingWords - устанавливает флаг НАЧАЛО НОВОГО СЛОВА и увеличивает счетчик слов.
    Если символ <,>,&,| то просто выход из функции.
    При получении исключительно пробела очищает массив возможных слов
 AddWord - В случае кол-ва слов > 0(т.е. одно слово уже точно есть) ищет подходящие
    ИМЕНА ФАЙЛОВ
    В ином случае ищет подходящие КОМАНДЫ

RecordSymb - записывает символ в string(строка хранящая команду)
